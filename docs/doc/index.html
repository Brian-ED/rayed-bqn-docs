Rayed-bqn documentation

Working BQN library for making games. Not stable, though usable. Breaking changes, when intentional, will be explained in the BQN forum.

Rayed-bqn's documentation describes what features it has, how to use them, and why they were chosen. The rayed-bqn documentation will avoid talking about bqn specifics like primitives, for those you may refer to the (BQN documentation)[https://mlochbaum.github.io/BQN]
